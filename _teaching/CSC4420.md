<!-- Description -->
<div class="cs-section">
<div class="cs-section-title">Course Description</div>
<p style="font-size:0.93em;color:#333;line-height:1.7;">
The operating system is an essential software layer for various computing devices. This course covers modern operating systems' basic components and design principles - including <strong>process &amp; thread</strong>, <strong>CPU scheduling</strong>, <strong>memory management</strong>, <strong>file systems</strong>, and more. Students also gain hands-on programming experience using Linux system calls and design and implement a cloud-based file system.
</p>
</div>
