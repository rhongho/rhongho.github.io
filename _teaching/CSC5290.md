<!-- Description -->
<div class="cs-section">
<div class="cs-section-title">Course Description</div>
<p style="font-size:0.93em;color:#333;line-height:1.7;">
This course explores broad security topics in the areas of network and operating systems. It focuses on providing <strong>hands-on experience</strong> leveraging various security tools, helping students understand real-world security threats. The course covers both <strong>offensive and defensive methods</strong> in a laboratory environment. Students are expected to complete lab assignments using real-world exploits and defense tools.
</p>
</div>

    <div class="obj-card-desc">Apply defense tools and mitigation strategies to protect networked systems from attacks.</div>

<!-- Topics -->
<div class="cs-section">
<div class="cs-section-title">Topics</div>
<div class="topic-grid">
  <div class="topic-card">
    <div class="topic-card-header">Network Security</div>
    <ul>
      <li>Network scanning and enumeration</li>
      <li>Traffic analysis and sniffing</li>
      <li>Intrusion detection systems</li>
      <li>Firewall and ACL design</li>
    </ul>
  </div>
  <div class="topic-card">
    <div class="topic-card-header">OS Security</div>
    <ul>
      <li>Linux privilege escalation</li>
      <li>System call auditing</li>
      <li>File system security</li>
      <li>Process isolation</li>
    </ul>
  </div>
  <div class="topic-card">
    <div class="topic-card-header">Offensive Techniques</div>
    <ul>
      <li>Vulnerability exploitation</li>

      <li>Social engineering awareness</li>
      <li>Web application attacks</li>
    </ul>
  </div>
  <div class="topic-card">
    <div class="topic-card-header">Defense &amp; Forensics</div>
    <ul>
